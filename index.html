Clustering & Outlier Detection from Scratch

Built for my CS634 Data Mining final project, this Python-based application performs unsupervised clustering and outlier detection entirely from scratch. I generated synthetic 3D datasets, removed statistical outliers using a k-nearest neighbor distance method, and then applied both K-Means and Hierarchical Agglomerative Clustering (HAC) with multiple linkage strategies. Each algorithm's results were evaluated using the Silhouette Coefficient to measure cluster quality. The program logs execution time, clustering effectiveness, and dynamically determines the best method for each dataset.
